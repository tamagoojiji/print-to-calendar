# プライバシーポリシー

**サービス名: print-to-calendar（以下「本サービス」）**
制定日: 2026年6月2日

BridgeSquare（以下「当方」）は、本サービスにおける利用者の個人情報および予定情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

---

## 1. 取得する情報

本サービスは、以下の情報を取得します。

| 取得する情報 | 取得方法 | 備考 |
|---|---|---|
| メールアドレス | 決済（Stripe）を通じて取得 | ライセンス発行・問い合わせ対応のため |
| Googleアカウント情報（メールアドレス・アカウント識別子） | Googleカレンダー連携（OAuth）時 | カレンダー連携先の特定のため |
| Googleカレンダーへのアクセストークン（リフレッシュトークン） | Googleカレンダー連携（OAuth）時 | **暗号化して保管**。予定の自動登録に使用 |
| ライセンスキー | 利用者の入力 | **平文では保存せず、ハッシュ化して保管** |
| アップロードされた予定プリント画像 | 利用者の画像アップロード時 | 予定抽出のために一時的に処理。**長期保存しません**（第4条参照） |
| 抽出・保存された予定データ（予定名・日時・場所・メモ等） | 画像解析および利用者の確認・保存時 | 予定の管理・カレンダー登録のため |
| 利用ログ（読み取り回数・利用日時・エラー情報等） | 本サービスの利用時 | 利用回数の管理・不具合対応のため |

## 2. 利用目的

取得した情報は、以下の目的のために利用します。

1. 予定プリント画像から予定情報を抽出し、利用者に提示するため
2. 抽出した予定を利用者のGoogleカレンダーに登録・更新・削除するため
3. ライセンスの有効性確認、利用期間・利用回数の管理のため
4. 決済処理、購入者へのライセンスキー発行のため
5. お問い合わせ対応およびサポートの提供のため
6. 不具合の調査・改善、不正利用の防止のため

## 3. 第三者提供・外部サービスへの委託

本サービスは、上記の目的の達成に必要な範囲で、以下の外部サービスを利用します。利用者の情報の一部は、これらのサービスの提供事業者に提供・委託されます。

| 外部サービス | 提供事業者 | 取扱う情報 | 目的 |
|---|---|---|---|
| Gemini API | Google LLC（米国） | アップロード画像 | 画像からの予定抽出（AI解析） |
| Google Calendar API / OAuth | Google LLC（米国） | Googleアカウント情報・予定データ | カレンダーへの登録・更新・削除 |
| Stripe | Stripe, Inc.（米国） | メールアドレス・決済情報 | 決済処理 |
| GitHub Pages | GitHub, Inc.（米国） | アクセス情報 | アプリ（PWA）の配信 |
| サーバー（VPS） | Contabo GmbH（ドイツ） | 本ポリシーに定める保存対象情報 | バックエンドの稼働・データ保管 |
| Discord（エラー通知） | Discord Inc.（米国） | エラー種別・内部ID等の技術情報のみ | 同期エラーの検知・対応 |

**Discordへのエラー通知について**: 同期エラーの調査のため、エラー内容を当方の通知チャンネルに送信しますが、**アップロード画像や、画像に含まれる氏名・学校名などの個人情報は送信しません**。送信するのは予定名・処理種別・内部識別子・エラー種別などの技術情報に限られます。

### 外国にある第三者への提供（越境移転）

上記のとおり、本サービスは米国（Google・Stripe・GitHub・Discord）およびドイツ（Contabo）に所在する事業者のサービスを利用します。利用者は、本ポリシーに同意し本サービスを利用することにより、これらの外国にある第三者へ個人情報が提供されることに同意するものとします。各国の個人情報保護制度や各事業者の措置については、各事業者のプライバシーポリシーをご確認ください。

## 4. 画像の取扱い（重要）

- アップロードされた予定プリント画像は、**予定情報を抽出する目的のためにのみ**、Gemini API（Google LLC）へ送信されます。
- 当方のサーバーやデータベースに、**画像そのものを長期保存することはありません**。
- 予定プリントには、お子さまの氏名・学校名など第三者の情報が写り込む場合があります。本サービスはこれらを保存対象とせず、AIが抽出した結果は**必ず利用者ご自身が確認のうえ保存**する仕組みとしています。
- 撮影・アップロードする画像は、ご自身または保存・登録について同意を得た範囲のものに限定してください。

## 5. 保有期間

- 予定データ・Google連携情報・ライセンス情報は、利用者がライセンスを利用している間、または当方が利用目的の達成に必要な期間、保有します。
- Google連携を解除した場合、当該リフレッシュトークンは無効化されます。
- 利用者からの削除のお申し出があった場合、法令に基づき保有が必要な場合を除き、合理的な期間内に削除します。

## 6. 安全管理措置

- Googleカレンダーへのアクセストークン（リフレッシュトークン）は暗号化して保管します。
- ライセンスキーは平文で保存せず、ハッシュ化して照合します。
- Gemini APIキー等の重要な認証情報はサーバー側でのみ管理し、アプリ（フロントエンド）には含めません。
- 通信は暗号化（HTTPS）により保護します。

## 7. 利用者の権利

利用者は、当方が保有する自己の個人情報について、開示・訂正・利用停止・削除等を請求することができます。ご請求は第9条の窓口までご連絡ください。本人確認のうえ、法令に従い合理的な期間内に対応します。
